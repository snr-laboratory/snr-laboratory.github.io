# snr-laboratory.github.io

Public website for The SNR Laboratory (snr-lab.org). The site is built with Pandoc and a custom HTML template, not Jekyll.

## Build & Serve
- `make build` generates the site into `docs/`.
- `make serve` builds and serves `docs/` at `http://localhost:8000`.
- `make clean` removes the generated `docs/` directory.
- `PANDOC=/path/to/pandoc make build` overrides the Pandoc binary if needed.

## Project Structure
- `content/` Markdown source pages (Pandoc). Each page renders to `docs/<slug>/index.html`.
- `theme/` Pandoc template and shared partials (`template.html`, `nav.html`, `footer.html`).
- `filters/` Lua filters (e.g., marginalia/commentary handling).
- `assets/` static files (CSS/JS/images), copied to `docs/assets/` on build.
- `docs/` generated site output for GitHub Pages.

## Notes
- Do not edit `docs/` by hand; it is generated by Pandoc.
- Update navigation in `theme/nav.html` when adding or renaming pages.
